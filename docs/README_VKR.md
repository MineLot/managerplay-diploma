### Цель и задачи
Цель: разработать прототип мобильного приложения «ManagerPlay» — сборника мини-игр для обучения менеджеров.

Задачи:
1. Провести анализ существующих решений  
2. Сформулировать требования к функционалу  
3. Спроектировать архитектуру и интерфейс  
4. Реализовать 3 мини-игры: «Кейс-битва», «День менеджера», «7 дней роста»  
5. Провести тестирование и оценку эффективности
